# UrbanPulse-AI
UrbanPulse AI is an advanced urban analytics platform that leverages NASA Earth observation data to build climate-resilient, sustainable cities.

## 📖 Overview

**UrbanPulse AI** is an advanced urban analytics platform that leverages NASA Earth observation data to build climate-resilient, sustainable cities. Our platform provides comprehensive infrastructure planning, climate risk assessment, and data-driven solutions for urban development.

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- pip package manager

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/your-username/urbanpulse-ai.git
cd urbanpulse-ai
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Run the application**
```bash
streamlit run app.py
```

4. **Access the app**
```
http://localhost:8501
```


### Satellite Missions Used
- **Landsat 8/9** - Urban expansion and land use change
- **MODIS** - Temperature patterns and heat islands
- **VIIRS** - Nighttime lights and urban activity
- **GRACE-FO** - Water resources and groundwater
- **SEDAC** - Population density and infrastructure
